This game can run locally and has been tested to work on both keyboard, mouse and mobile touch screen devices. However touch screen devices have a limitation where you can't access the achievements or help menu as they rely on keyboard presses from listen events rather then a text box and no virtual keyboard is triggered for the mobile device.

Key features

Increased level difficulty, each coin produces a new rock which adds to the list of objects to avoid.

Local achievements! all achievements are stored locally and cover a wide range of different datasets.

Pausing the game with the 'p' button.

Muting the background noise with the button located external to the canvas.

Support for keyboard and mouse controls.
